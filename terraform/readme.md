# How to start
## Build
docker-compose build

## Start
docker-compose up

## Api
http://localhost:5000/pumpkins - GET/POST (json)
http://localhost:5000/pumpkins/<id>

## Terraform
cd terraform
./bin/terraform init
./bin/terraform apply
